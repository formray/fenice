# FENICE 3D World
## Ritmo Operativo Team (Giuseppe + Codex + Claude)

## Cadence
1. Lunedi 20 min: planning e lock scope.
2. Martedi-Giovedi: execution async con daily update su board.
3. Venerdi 30 min: demo/review e decisioni per sprint successivo.

## Regole di collaborazione
1. Nessuna modifica strategica senza update al Decision Log.
2. Ogni task in progress deve avere owner e ETA.
3. Ogni blocker oltre 24h va esplicitato nel board.
4. Prima demo funzionante, poi ottimizzazione.

## Output minimi settimanali
1. Sprint board aggiornato.
2. Decision log aggiornato.
3. Demo artifact (video o live).
4. Lista rischi aggiornata.
