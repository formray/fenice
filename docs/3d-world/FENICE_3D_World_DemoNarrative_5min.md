# FENICE 3D World
## Demo Narrative - 5 minuti

Data: 2026-02-21
Target: stakeholder tecnici/prodotto

## 1. Setup (30s)
1. Apri la city view.
2. Spiega in una frase: "Questa citta' rappresenta la tua API reale".

## 2. Orientamento (45s)
1. Evidenzia un distretto (service/tag).
2. Mostra che ogni building e' un endpoint.
3. Mostra legenda colori per metodo HTTP.

## 3. Drill-down endpoint (60s)
1. Clic su building.
2. Side panel con path, method, schema, auth.
3. Mostra relazione con altri endpoint tramite edges.

## 4. Decisione operativa simulata (75s)
Scenario:
"Dobbiamo aggiungere un endpoint per reset password"

1. Identifica area coerente della city.
2. Mostra gap strutturale (building mancante).
3. Spiega come questa vista accelera decisione di design.

## 5. Visione prossima fase (60s)
1. Anticipa overlay realtime: health, latenza, errori.
2. Anticipa AI Builder PR-first:
   - prompt
   - proposta diff
   - test
   - PR

## 6. Chiusura (30s)
1. Recap valore:
   - comprensione sistema piu' rapida
   - decisioni architetturali piu' chiare
   - ponte naturale verso AI-assisted development
2. Call to action: "Approvazione M2 realtime overlay".

## 7. Checklist pre-demo
1. Build client pronta.
2. OpenAPI caricata correttamente.
3. Camera controls testati.
4. Nessun warning bloccante console.
5. Piano fallback (video registrato) disponibile.
